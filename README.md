🍦 Ice Cream Paradise – Responsive Website

Ice Cream Paradise is a modern and visually appealing front-end website designed for an ice cream brand. It focuses on clean UI design, smooth layout, and fully responsive behavior across all devices.

---

✨ Features:
- Modern and attractive landing page design
- Fully responsive layout (mobile, tablet, desktop)
- Eye-catching hero section
- Product showcase sections
- Smooth navigation and user-friendly interface
- Clean and organized UI using Bootstrap and CSS

---

🛠️ Technologies Used:
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Google Fonts
- Font Awesome Icons

---

Project Purpose:
This project was created to improve front-end development skills and practice responsive web design. It demonstrates layout structuring, styling consistency, and basic interactivity for real-world websites.
